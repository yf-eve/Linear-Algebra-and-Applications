# Linear Algebra & Signal Processing Lab

This repository explores the intersection of **Linear Algebra** and **Signal Processing**. It documents my implementation of various algorithms for 1D (Audio) and 2D (Image) signal restoration and analysis.

## 📁 Repository Structure
- **[`audio_signal/`](./audio_signal)**: Time-frequency analysis and discrete signal processing.
- **[`image/`](./image)**: Inverse problems in image restoration (Deblurring, Inpainting).
